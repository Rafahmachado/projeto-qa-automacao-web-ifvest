# Automação de Testes Web – IFVEST

Projeto de automação de testes web utilizando Cypress,
baseado em fluxos reais testados durante experiência prática
em uma plataforma educacional.

## Objetivo
Demonstrar conhecimentos iniciais em automação de testes,
configuração de ambiente, execução de cenários e geração
de relatórios automatizados.

## Funcionalidade Automatizada
- Login com credenciais válidas

## Tecnologias Utilizadas
- Cypress
- JavaScript
- Node.js
- Mochawesome (relatórios)

## Estrutura do Projeto
- Testes E2E localizados em `cypress/e2e`
- Configuração do Cypress em `cypress.config.js`
- Relatórios gerados automaticamente após a execução

## Execução dos Testes
1. Instalar as dependências:
- npm install

3. Executar os testes e gerar relatório:
- npm run test:full

## Observações Importantes
- Projeto de estudo e portfólio
- Credenciais utilizadas são de teste e não representam
usuários reais
- O projeto encontra-se em evolução, com melhorias futuras
como refatoração para Page Object e novos cenários de teste
